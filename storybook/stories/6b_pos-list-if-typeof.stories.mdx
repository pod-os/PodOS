## pos-list if-typeof

Renders all things of a given type found in the current store, with a custom
template. The related resources can optionally be fetched before rendering.

<Canvas withSource="open">
  <Story name="pos-list if-typeof">
    {({ uri, type, fetch }) => html`
      <pos-resource uri=${uri}>
        This resource is loaded so there is something in the store.
      </pos-resource>
      The template is repeated for each item:
      <pos-list all-from-store if-typeof=${type} ?fetch=${fetch}>
        <template> :-) </template>
      </pos-list>
    `}
  </Story>
</Canvas>
